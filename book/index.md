# Parallel Critiques: Analyzing Rhetorical Extremism

This interactive book presents a data-driven analysis of textual and ideological overlap between a Jordan Peterson YouTube lecture and Anders Behring Breivik's manifesto.

- 16% semantic similarity (TF-IDF)
- ~75% conceptual overlap (thematic clustering)
- Shared genealogy (Marx → Frankfurt School → identity politics)
- Divergence: abstract critique vs. concrete threat identification

Use the "launch" buttons (top-right) to run pages in Binder, or click the rocket to enable Thebe for in-browser code cells when available.

```{admonition} How to use
- Read the chapters sequentially or jump to a section.
- When code cells are present, you can execute them live in Binder.
```
